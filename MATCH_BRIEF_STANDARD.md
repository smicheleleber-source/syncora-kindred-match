# Match Brief Standard

## Purpose

The Match Brief is the standardized output of the Discovery Engine.

Its purpose is to provide providers with a concise summary of client needs and improve matching quality.

## Universal Match Brief Fields

### Request Information

* Request Type
* Vertical
* Category
* Subcategory

### Priority

* Urgency
* Complexity

### Geography

* City
* County
* State
* Service Area

### Budget

* Budget Range
* Funding Source (if applicable)

### Timeline

* Target Date
* Court Date (if applicable)
* Event Date (if applicable)

### Summary

Plain-language summary of the user's needs.

### Discovery Responses

Structured responses gathered during intake.

## MVP Rule

Every match generated by SyncoraConnect should create a Match Brief.

Providers receive the Match Brief after a match is generated.

The Match Brief becomes the primary communication artifact between clients and providers.
